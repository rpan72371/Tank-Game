My first proper project, TANK-GAME(NAME PENDING) created for the ECSE design competition 2026 UOA!

Controls:
"W" Move up
"S" Move down
"A" Move left (FREE MOVE ONLY)
"D" Move right (FREE MOVE ONLY)

"Scroll wheel" Aim cannon
"LMB" Fire cannon
"RMB" Activate powerup

Special Toggles:
"F12" Toggle CRT shader
"Shift+F12" Toggle CRT curvature
"Alt+F12" Toggle vertical orientation
"F10" Toggle free move
